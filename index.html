Whale Yatzhee Rules

Player with most points wins!

  • Player 1 goes first
  • Click the "Roll" button to roll a five virtual six-sided die
  • Values 1-6 can be claimed for the respective point value once per game or rerolled to go for another set
  • To end turn, either roll the maximum of three times or select "End Turn" button if you are happy with your selection- this will add the "Roll Total" to the player's score
  • Game is over after each player has played six turns, selecting each number once, highest total wins
  • Have fun! :D
